# Capstone Project: Deploying Microservices

This is the final capstone repository for the IBM Cloud Code Engine and Red Hat OpenShift microservices lab.

## Project Overview
- **Pictures Service:** A Flask microservice intended for deployment to IBM Code Engine.
- **Songs Service:** A Django microservice intended for deployment to Red Hat OpenShift.
- **Cartoons Service:** A secondary microservice used for Docker and local Kubernetes testing.

## Objectives
1. Deploy the Pictures service to IBM Cloud Code Engine.
2. Deploy the Songs service to Red Hat OpenShift.
3. Migrate databases using Django ORM.
4. Containerize applications using Docker.
5. Orchestrate using Kubernetes.

## Technologies
- Python, Flask, Django
- Docker
- Kubernetes
- IBM Cloud Code Engine
- Red Hat OpenShift
